About This Course

Welcome to WDD 131 - Web Design and Development. This course covers the fundamentals of creating modern, responsive websites using HTML, CSS, and JavaScript.

Students will learn to create semantic markup, apply modern CSS techniques, and add interactive functionality with JavaScript.

Course Projects

Project 1: Basic HTML Structure

Creating semantic HTML documents with proper structure and accessibility.

Project 2: CSS Styling

Applying modern CSS techniques including Flexbox and Grid layouts.

Project 3: JavaScript Interactivity

Adding dynamic behavior and user interaction with JavaScript.
